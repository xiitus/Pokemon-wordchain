# Pokemon-wordchain⚡️⛓
## Install
Requires python3.8+ (3.8, 3.9, 3.10, 3.11)

```
cd ~
cd Desktop
git clone https://github.com/xiitus/Pokemon-wordchain.git
cd Pokemon-wordchain
python3 main.py
```

### then, you can play Pokemon-wordchain⚡️⛓
### wanna quit? and if so, you can press ^C(Control + C).
